# 고1 수학 50일 완성 - Digital Garden 🌱

Obsidian Digital Garden을 활용한 고1 수학 학습 노트 시스템

📚 **웹사이트**: https://high-school-math-garden.vercel.app

## 프로젝트 소개

고1 수학을 체계적으로 학습하기 위한 디지털 노트 시스템입니다.  
주 2회 학습으로 8개월간 45개 강의를 완주하는 것이 목표입니다.
